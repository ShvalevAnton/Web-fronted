Управление CSS из JavaScript

Описание задания:

Эта страница демонстрирует изменение стилей элемента с помощью JavaScript:

  • Создана функция ChangeStyle(), которая изменяет цвет, размер, добавляет подчеркивание и выделение полужирным шрифтом текста
  • Функция применяется к элементу с id="change"
  • Функция вызывается при наведении курсора на ссылку
